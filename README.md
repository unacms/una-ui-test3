# UNA e2e tests using Playwright

Install depedences:
```bash
npm install
```

Install browsers:
```bash
npx playwright install
```

Run tests:
```bash
npm run test
```

Run tests interactively:
```bash
npm run test:headed
```

Debug tests:
```bash
npm run debug
```

Run tests code generator:
```bash
npm run codegen
```
